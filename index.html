CSS BOX MODEL

A box model is basically a box that wraps around every HTML element
It allows you to add a border around elements and to define spaces between elements

Every box is composeed of four parts or rather yet properties :

Margin

Being the outermost layer extends the border area to include an empty area used to separate the element from other elements
Its dimensions are the margin-box width and the margin-box height

Border

This is right after the margin bounded by the border edge, extends the padding area to include the element's borders.
Its dimensions are the border-box width and the border-box height.

Padding

Extends the content area. Its dimensions are the padding-box width and the padding-box height.

Content

contains the "real" content of the element, such as text, an image, or a video player.
Its dimensions are the content width and the content height.

Calculating dimensions

Consider a box model with a width of 400px, padding of 30px, margin of 20px and a border of 2px

When finding the dimensions of a box model:


width=border + padding + content width
height=border + padding + content height
